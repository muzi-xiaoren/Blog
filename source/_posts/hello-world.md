欢迎来到我的博客！这是用 [Hexo](https://hexo.io/) + [Fluid](https://github.com/fluid-dev/hexo-theme-fluid) 主题搭建的个人站点。

## 怎么写新文章

在博客源码目录运行：

```bash
hexo new "文章标题"
```

文章会生成在 `source/_posts/` 下，用 Markdown 编写即可。

## 怎么发布

写完后运行：

```bash
hexo clean && hexo generate && hexo deploy
```

几分钟后刷新 https://muzi-xiaoren.github.io 就能看到更新。
